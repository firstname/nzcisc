User Guide

This system can analysis scores of eaxm or questinare. And it's very easy to use.

Before use

You need to know your purpose, it's to say, what you want to get through analysis, and it will be better if you know exactly some tables or charts you want. And these decide what kind of data you will input. Often seen file types are as blow:

  • Exam score data file
  • File about participants
  • File about test items

How to use

What you need to do is so easy, only THREE steps: Choose or upload data file >> Define groups or labels of your data >> Computer run and you'll see the result. And all the computational job is done by computer,what you would do is less. Come on, just do it!

  • Upload file in cooresponding format.
  • Define groups or labels of data
  • Check and download the results










File Templates

First of all, make sure of your purpose. Then choose the right file format and download the template, input all your data as exactly in the templates.

Exam score
